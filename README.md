# Tesla — Landing Page (Clone)
 
A pixel-styled, front-end clone of the Tesla Automotive Company website, built with pure **HTML** and **CSS**. The project recreates the look and feel of the official Tesla homepage — as a static, responsive layout.
 
🔗 **Live Demo:** [amalshad.github.io/Tesla](https://amalshad.github.io/Tesla)
 
---
 
## Overview
 
This project was built to practice real-world front-end layout skills: multi-section pages, flexbox/grid-based positioning, custom typography, and recreating a well-known brand's visual identity using only HTML and CSS.

## Project Structure
 
```
Tesla/
├── index.html          # Main page markup
├── style.css            # All styling
├── logo.png / llogo.png # Brand logos
├── *.svg / *.png / *.webp / *.jpg   # Icons & product/banner images
└── README.md
```
